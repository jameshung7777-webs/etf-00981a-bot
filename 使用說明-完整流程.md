# 00981A 完整流程說明

## 本地一次執行（一鍵跑完）

### 方法一：雙擊 `run_once.bat`
- 自動執行：抓取 → 儲存 JSON → 比較昨日 → 發送 Telegram
- 產生的檔案：
  - `holdings_data.json`（最新，供下次比較用）
  - `holdings_data_YYYY-MM-DD_HHMM.json`（帶日期時間的歷史檔）

### 方法二：命令列
```bash
python main.py --now
```

## GitHub 自動運行

- **排程**：每天台灣時間 18:00 自動執行
- **流程**：抓取 → 儲存 JSON → 發送 → **提交 JSON 到 repo**
- **每日持股檔**：會自動 commit 到 GitHub，方便查看歷史與明日比較

## 必要設定

1. **Telegram**：在 GitHub Secrets 設定 `TELEGRAM_BOT_TOKEN`、`TELEGRAM_CHAT_ID`
2. **本地**：`pip install -r requirements.txt`，並設定 `config.py`
